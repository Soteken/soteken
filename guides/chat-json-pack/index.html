JSON от чата

Как сделать пак через обычный чат

Этот способ нужен, если у вас нет Codex или MCP. Достаточно обычного ChatGPT, Claude, DeepSeek или другого чата.

  1. Откройте чат

    Подойдёт любой чат, который умеет писать длинный текст и создавать JSON: ChatGPT, Claude, DeepSeek и похожие сервисы.

  2. Отправьте инструкцию

    Скопируйте этот текст и отправьте его в чат перед своим запросом:

    Ты должен подготовить JSON-файл для импорта в SotekenGamePack.
Сначала уточни у меня:
1. Сколько раундов нужно?
2. Сколько тем в каждом раунде?
3. Сколько вопросов в каждой теме?
4. Какая стоимость вопросов?
5. Для какой аудитории делаем пак?
6. Какие темы точно нужны и какие темы нельзя использовать?

После уточнений верни готовый файл soteken_pack.json.
Не пиши объяснения вместо файла. Нужен именно JSON для импорта.

Структура:
{
  "name": "Название пака",
  "author": "Автор",
  "rounds": [
    {
      "name": "Раунд 1",
      "themes": [
        {
          "name": "Тема",
          "questions": [
            {"price": 100, "text": "Вопрос", "answer": "Ответ"}
          ]
        }
      ]
    }
  ]
}

  3. Попросите нужный пак

    Например:

    Сделай пак на 3 раунда, по 5 тем, по 7 вопросов.
Пак для друзей 20 лет. Темы: мемы, игры, кино, музыка, YouTube.

  4. Скачайте или скопируйте JSON

    Идеально, если чат вернёт файл `soteken_pack.json`. Если файла нет, можно скопировать JSON-код вручную в текстовый файл.

  5. Импортируйте в редактор

    Откройте SotekenGamePack и нажмите `JSON от чата`. После импорта останется добавить картинки, аудио и видео.

Зачем это нужно

Чат создаёт структуру: раунды, темы, вопросы и ответы. Редактор нужен для медиа, проверки и экспорта готового `.siq`.
