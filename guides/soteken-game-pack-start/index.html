Как скачать редактор

Эта инструкция для обычного пользователя SotekenGamePack. Не нужно скачивать исходный код и разбираться в GitHub.

Важно: если готовая сборка ещё не опубликована, на странице релизов может быть пусто. В таком случае нужно дождаться релиза или запускать проект из исходников.

  1. Откройте страницу скачивания

    Перейдите на страницу релизов проекта. Это место, где лежат готовые архивы для пользователей.

    Открыть страницу релизов

  2. Найдите последний релиз

    Нужен самый верхний релиз. Обычно рядом написано `Latest` или он просто находится первым.

  3. Скачайте архив из Assets

    На странице релиза найдите блок `Assets`. Скачивайте архив с готовой Windows-сборкой, а не исходный код.

    Не нужно нажимать зелёную кнопку `Code`. Она скачивает файлы для разработчиков.

  4. Распакуйте архив

    Создайте папку в удобном месте, например на рабочем столе или в документах, и распакуйте туда архив.

  5. Запустите приложение

    Откройте файл `SotekenGamePack.exe`. После запуска редактор сам откроется в браузере.

Что дальше
