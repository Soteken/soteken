Готовые Windows-сборки будут публиковаться в релизах. Это страница для обычного пользователя, которому не нужен исходный код.

Если релизов ещё нет: страница GitHub может быть пустой. Значит готовая сборка ещё не опубликована.

  1. Откройте релизы

    Релизы — это место, где лежат готовые архивы приложения.

    Открыть релизы

  2. Скачайте архив из Assets

    Не нажимайте зелёную кнопку `Code`: она скачивает исходники для разработчиков.

  3. Распакуйте и запустите

    Распакуйте архив в удобную папку и откройте `SotekenGamePack.exe`.
